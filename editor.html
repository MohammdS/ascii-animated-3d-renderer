JSON format

Use a non-empty array of [x, y, r, g, b] points. Coordinates can use any scale; the editor fits the view automatically. RGB values must be between 0 and 255.

Drag to paint or erase. Right-click temporarily removes points regardless of the selected tool. Export downloads a new JSON file and never overwrites the source.
